Detailed Description

Implementation of the Parser class and related functionality.

This file contains the implementation of the Parser class, which is responsible for managing the parsing of log messages using registered body parsers for specific encodings. It processes structured log messages and produces a ParseResult containing the parsed messages and any errors encountered.
