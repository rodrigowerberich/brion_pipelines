Detailed Description

Message class representing a log message in a pipeline.

This class encapsulates the details of a log message, including its pipeline ID, message ID, body, and next ID.

The encoding is not included in this class, as it is this version of the message should already contain the decoded body.

It provides some utility methods e.g. comparison and output formatting.
